Formatters for Go lexer.
Supported the following formatters from this list:
https://github.com/life4/awesome-go-code-formatters 
They all must be found in PATH.

- gofmt
- goimports
- goreturns
- gofumpt
- golines

Tool 'golines' supports additional command line arguments. Write them
to file [CudaText]/settings/plugins.ini:

  [golines]
  params=--max-len=300
